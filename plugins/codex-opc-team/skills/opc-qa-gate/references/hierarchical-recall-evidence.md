# Hierarchical recall QA evidence

Require evidence for authority, safety, quality, cost and degradation together:

| Gate | Evidence |
|---|---|
| Authority | Every packet item has canonical relative path, current HEAD commit/hash, approved status and explicit scope |
| Governance | #7 hard-filter order; unresolved conflicts have two citations and no bodies |
| Privacy | Trace contains no body/summary, credentials, raw chat, Hook/session/home data or unauthorized private knowledge |
| Storage | Preview zero write; private contained atomic build/delete; link/parent/oversize/duplicate rejection |
| Degradation | missing/stale index and provider missing/disabled/timeout/error/disagreement remain usable |
| Evaluation | Same fixture for flat/treatment; zero safety acceptance; strict finite metrics; correct superior/not-superior claim |

Run both baseline verifies. Treat the versioned wall-clock latency artifact as nondeterministic local evidence, not byte-reproducible or cross-machine proof.
